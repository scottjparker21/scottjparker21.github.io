E-Commerce Website


One of the largest projects I've done so far is a mock ecommerce website. For this site I used a LAMP stack platform(Linux/Apache/MySQL/PhpMyAdmin). I chose to do a bar/restaurant supply site. The site features user login and registration, full admin crud, and functioning cart. The live search functionality is handled with Javascript. Almost everything on the site is populated using information from the database making it lightweight and flexible. The site is also fully responsive and mobile freindly.
